# Entén l'IRPF - Catalunya

Aplicació React + Vite per visualitzar i comparar trams d'IRPF a Catalunya.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm start`
